Principais formatações

Negrito / Destaque

Nessa frase temos um trecho em negrito. usando a tag <b> (não semântico).

Nessa frase temos um trecho em negrito usando a tag strong (semântico).

Itálico / ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semântico)

Aqui usamos um atalho ctrl + shift + p. digitamos abb, clica em wrap with abbreviation, digita a tag de interesse (p, i, em, b ou strong...)

Nessa frase temos um termo em itálico usando a tag em (semântica)

Texto marcado

Podemos criar uma frase marcada tipo um marca texto. utilizando a tag mark.

Outra opção é utilizando a css para alterar a cor do marca texto: Podemos criar uma frase marcada com cores diferentes.

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno usando as tags big e smmall.

Texto deletado

Podemos marcar uma frase como excluída para indicar que ela deve ser lida, mas não considerada. Para isso utilizamos a tag del.

Texto inserido

Podemos marcar uma frase como inserida para dar ênfase e indicar que ela foi adicionada depois. Para isso utilizamos a tag ins

Texto subscrito

Para inserir uma parte do texto subscrito tipo ex: H2O utilizamos a tag sub.

Texto sobrescrito

Para inserir uma parte do texto sobrescrito tipo ex: 22 + 2 utilizamos a tag sup
